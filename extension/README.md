# EduFlow 智慧树卡片注入 - Chrome 插件

任意页面均可打开扩展；按六步流程完成「选择文件 → 选框架 → 选人设 → 编辑评价项 → 生成卡片 → 注入平台」。无需配置 Cookie/课程/任务/节点 ID。

## 使用前提

- **默认**：扩展使用 [eduflows.cn](https://eduflows.cn) 后端，无需改地址。
- **API Key**：解析与生成需 LLM API。首次使用请在侧边栏顶部「**LLM 设置**」模块中填写 API Key、选择供应商，点击「保存配置」。可点击「测试连接」验证配通。
- **本地部署**：若本地运行网站，在「LLM 设置」中把后端地址改为 `http://localhost:端口`。
- 注入前需打开**智慧树能力训练配置页**（某门课的卡片编辑页），扩展会自动检测当前标签页是否为该页面。

## 安装

1. Chrome 地址栏输入 `chrome://extensions/`，打开「扩展程序」。
2. 开启右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」，选择本项目的 `extension` 目录。

## 六步流程

1. **选择文件**：拖入或选择 .md / .docx / .pdf 剧本。后端自动解析、分幕、生成 trainset 和学生角色（人设），并提取任务名称、描述和评价项。
2. **选择生成框架**：从下拉选择卡片生成框架（如 dspy）。
3. **选择学生角色**：从预设或本次生成的人设中选择；可查看并编辑 YAML 内容后保存。
4. **评价项**：从文档中自动提取的评价项会在此展示，可增删改每项的名称和分值。
5. **生成卡片**：点击「生成卡片」得到 Markdown（含评价项章节）；可在文本框内编辑后再执行注入。
6. **注入平台**：确保当前标签页为智慧树能力训练配置页，点击「注入平台」。注入包含节点创建、连线设置、评价项创建和任务配置，完成后刷新画布即可看到。

## LLM 设置模块

侧边栏顶部的独立设置区域，管理生成链路所需的 LLM 配置：

- **供应商/模型类型**：DeepSeek / 豆包 / OpenAI 兼容
- **API Key**：填写后点击保存；已保存的 Key 脱敏展示
- **Base URL / Model**：跟随预设自动填充，也可手动覆盖
- **测试连接**：发送最小请求验证配置是否可用
- **恢复预设**：将 Base URL 和 Model 重置为当前供应商默认值
- 摘要态始终在标题栏显示当前配置状态

## 文件说明

- `manifest.json` - 插件配置与权限（含 sidePanel）。
- `background.js` - 认证、智慧树 API、当前标签页检测。
- `sidepanel.html` / `sidepanel.js` / `sidepanel.css` - 侧边栏六步流程 UI 及 LLM 设置模块。
- `content.js` - 仅在智慧树配置页运行，响应「注入平台」并执行节点/连线/评价项注入。
