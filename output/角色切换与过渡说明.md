# 角色切换与过渡说明

基于测试日志与评估报告（evaluation-report-2026-02-02T09-44-34.md）的结论：

## 现象

1. **第5轮**：接诊（母亲）→ 查体（责任护士）时，学生上一句还是母亲在说孕期用药，下一句直接变成“你说得对，心率和呼吸都明显超出了……”，**中间没有过渡语**，承接生硬。
2. **第13轮**：术后护理（母亲）→ 并发症识别（医生）时，直接变成“护士，患儿今天的情况你观察到了哪些异常？”，**缺少时间/场景铺垫**（如“术后第2天”）。
3. **第18轮**：并发症（医生）→ 出院宣教（母亲）时，出现了 B 卡旁白“经过及时识别和规范汇报……现在，请将注意力转向病房，患儿的母亲正带着许多关于回家后护理的疑问……”，但评估指出：**第三人称旁白打破角色扮演沉浸感**，更自然的方式是让新角色直接开口（如母亲说“护士，我们明天要出院了，回家要注意啥呀？”）。

## 原因归纳

- **平台可能不展示 B 卡**：跳转后直接进入下一张 A 卡，学生看不到 B 卡的过渡语。
- **B 卡偏旁白体**：即使展示，也是“您已经……现在请将注意力转向……”的教学口吻，不像剧情内的角色切换。
- **新 A 卡第一句缺少承接**：没有“好的，我们到病房了”“术后第二天了”等简短情境句，直接从新角色业务台词开始。

## 改进思路

1. **B 卡**：过渡语尽量**短、像场景提示**，少用“您已经……现在请……”；可改为一句情境句（如“病房里，责任护士已准备好查体报告。”），或明确要求“若平台不展示 B 卡，下一张 A 卡的第一句须含简短情境承接”。
2. **新 A 卡（角色切换后）**：在 Interaction 或 Context 中增加**开场第一句**要求：先一句**情境承接**（时间/场景/身份），再进入本角色第一句台词，例如：
   - 2A：先“好的，我们到病房了。这是刚拿到的查体数据。”再给体征与提问。
   - 4A：先“术后第二天了，患儿有些新情况。”再“护士，患儿今天的情况你观察到了哪些异常？”
   - 5A：母亲直接开场：“护士，我们明天要出院了，回家要注意啥呀？”（不用旁白总结，直接角色开口。）

**已完成的修改**（`cards_output_20260202_170046.md`）：

- **1B**：Output 改为简短场景句「病房里，责任护士已为患儿完成体格检查并备好报告……」；Context 注明若平台不展示则 2A 第一句须含情境承接。
- **2A**：Context 与 Interaction 明确「角色切换后第一句须先简短承接情境」（如「好的，我们到病房了」），再给体征与提问。
- **2B**：Output 改为简短「患儿病情可能出现新变化。接下来，患儿母亲会因宝宝吃奶时突然气促加重而向您呼救……」；Context 注明若平台不展示则 3A 第一句须是母亲呼救。
- **3A**：Interaction 开头强调「开场第一句仅输出母亲呼救的这一句台词」，不要输出括号、分支说明。
- **3B**：Output 改为「术后第二天，患儿出现新的异常情况。主治医生需要您结合监护数据……」；Context 注明若平台不展示则 4A 第一句须先点明时间与场景。
- **4A**：Context 与 Interaction 明确「角色切换后第一句须先点明时间与场景」（如「术后第二天了」），再以医生身份提问。
- **4B**：Context 注明避免长段第三人称旁白；Output 改为「病情稳定后，患儿即将出院。患儿的母亲会向您咨询……」；若平台不展示则 5A 第一句须以母亲身份直接开口。
- **5A**：Context 与 Interaction 明确「开场第一句须是母亲直接提问」（如「护士，我们明天要出院了，回家要注意啥呀？」），不用旁白。
